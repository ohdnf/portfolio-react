# React Portfolio

Vite + React + TypeScript 기반의 공개용 백엔드 포트폴리오입니다.

## Scripts

```bash
npm run dev
npm run build
npm run preview
```

## Deployment

GitHub Pages 배포 repository 이름은 `portfolio-react`를 기준으로 합니다. Vite `base`는 `/portfolio-react/`로 설정되어 있습니다.

## Content Policy

회사 private repository 원본 코드와 민감한 AWS 식별자는 포함하지 않습니다. 아키텍처와 데이터 스키마는 외부 공개 가능한 범위에서 핵심 구조 중심으로 재구성합니다.
